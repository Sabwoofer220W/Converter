# Сonverter USD to RUB
## Конвертер доллара к рублю и наоборот.  
Для инициализации проекта используйте команду.  
``` console 
npm install  
```  
Для запуска используйте:
``` console 
  npm start
```  
Либо:
``` console 
  nodemon index
```  
В браузере вводим http://localhost:5000/
